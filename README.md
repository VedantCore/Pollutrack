# Pollutrack - AI-Based Air & Dust Pollution Monitoring System

A modern, full-stack ready web application built with React, TypeScript, and Tailwind CSS for monitoring air quality and predicting pollution trends.

## 🚀 Tech Stack

- **Frontend**: React (Vite) + TypeScript
- **Styling**: Tailwind CSS (v4) + PostCSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Maps**: React-Leaflet
- **Router**: React Router DOM

## 🛠 Features

1.  **Modern Landing Page**: Glassmorphism UI, smooth entry animations, and interactive elements.
2.  **Smart Dashboard**: Real-time AQI tracking, pollutant breakdowns (PM2.5, PM10, CO, NO2), and trend analysis.
3.  **AI Prediction Module**: Visualization of predicted vs actual pollution levels with actionable insights.
4.  **Interactive Map**: Geospatial view of pollution data (Mock data focused on MIT ADT location).

## 📦 Installation & Setup

1.  Clone the repository (if not already in the project folder).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📂 Project Structure

- `src/components/`: Reusable UI components (Cards, Charts, Map).
- `src/pages/`: Main application pages (Landing, Dashboard).
- `src/data/`: Mock data for the application.
- `src/lib/`: Utility functions.
